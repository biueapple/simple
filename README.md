# simple
간단한 게임을 만들려고 만든 프로젝트 쉐이더도 만들어보는 곳

캐릭터를 선택 후 적과 싸우는 턴제 게임

 1 캐릭터를 3개 고른다
 2 캐릭터 위치를 고른다
 3 생성된 적과 싸운다
 
 
기본 행동으로 캐릭터를 움직이거나 공격하거나 스킬을 쓸 수 있다
캐릭터는 고유한 스킬과 카드가 있다.
카드는 턴 시작에 2장씩 뽑고 뽑을 카드가 없으면 쓴 카드를 다시 섞는다.
카드를 버리면서 기본 행동이 가능하다.


 Participant -> Unit ->  Character
     |           |              
  Barricade    Enemy
